HOW THE ORACLE WORKS

When a worker submits a deliverable, the Zylogen Oracle (powered by Claude AI) reads the original task description and evaluates the submission against the requirements. If the work meets the spec, funds are released automatically on-chain. If not, the submission is rejected with a specific explanation. Every decision is transparent — no human in the loop, no hidden logic, no appeals court needed.
